# Varianta comentată (folder separat)

Acest folder conține varianta de cod comentată manual (fără generator automat).
Comentariile trebuie să explice concret ce face codul și, unde e util, sintaxa folosită.

## Reguli de lucru

1. Fișierele originale din rădăcina proiectului rămân fără comentarii.
2. Comentariile se editează manual doar în fișierele din acest folder (`varianta_comentata/`).
3. Când modifici codul în varianta originală, aplică aceeași modificare în varianta comentată.
4. Când modifici varianta comentată, păstrează identic codul executabil față de original (diferența permisă este doar comentariul).
5. Evită comentariile generice; explică intenția logică și sintaxa importantă (`for`, `if`, destructuring, dict/array literal etc.).

## Fișiere incluse

- `serviciu_masuratori.py`
- `static/client_masuratori.js`
- `templates/panou_masuratori.html`
